Objective:
------------

To understand newly added types added in the collection framework in jdk5 with respect to concurrency.

How to run demo:
----------------------

This project contains 3 applications.

1. This application shows synchronized wrappers to make collection implementations thread-safe.
	
	java com.seed.test.collection.TestSynchronizedWrappers
	--------------------------------------------------------------------
	
2.	This application shows use of ArrayBlockingQueue.

	java com.seed.test.queue.TestArrayBlockingQueue
	-------------------------------------------------------------
	
3.	This application shows use of SynchronousBlockingQueue.

	java com.seed.test.queue.TestSynchronousQueue
